## Description

[Nest](https://github.com/nestjs/nest) project for Eventthai pre-interview test.

## Dependency Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
